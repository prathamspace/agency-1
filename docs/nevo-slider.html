ThemeVillain's exclusive

Nevo Slider

Setting up

Simply copy the markup from the features (or index) page.

Minimum requirements

To have a working slider, you'll need at least:

  • A wrapper (div) that contains the ".nevo-slider" class
  • A container (div) within that wrapper, using the ".slides" class
  • A slide element, which has the ".slide" class. These can be as many as you want. Beware: more slides will impact performance.
A simple example
Slide 1
Slide 2
Slide 3

div.nevo-slider
    div.slides.text-center
        div.slide.black-bg.light-text.section-size-3 Slide 1
        div.slide.lighter-bg.section-size-3 Slide 2
        div.slide.color-bg-1.section-size-3 Slide 3

As you can see, we use some helper classes here that adjust the size and colors.

Need more space?

Use the following classes on your slider.

  • .halfscreen
  • .fullscreen
